## Run Locally

Clone the project

```bash
  git clone https://github.com/Vikram-Hegde/Tasker
```

Go to the project directory

```bash
  cd Tasker
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
